## 🚀 GetStart

These instructions will allow you to get a working copy of the project on your local machine for development purposes.

### 🔧 Installation

Clone the project:

```
git clone https://github.com/LucasLuisBorges/portifa.git
```

install the dependencies

```
yarn install
```

Run the project:

```
yarn dev
```
